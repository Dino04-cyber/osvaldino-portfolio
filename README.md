# Osvaldino Jovete Portfolio v5

A React + TypeScript + Vite portfolio built around a data analytics pivot and operations intelligence positioning.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints, usually:

```text
http://localhost:5173
```

## Build for Netlify

```bash
npm run build
```

Netlify settings:

```text
Build command: npm run build
Publish directory: dist
```

## Add profile picture

Place a square image in `public/` and name it:

```text
profile.jpg
```

Recommended size: 900 x 900 px or larger.

## Edit projects

Update project content in:

```text
src/data/projects.ts
```

Use the case-study structure:

- Problem
- Build
- Data angle
- Proof
